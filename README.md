# urban_info_group4_final_proj
Final project repository for Group 4 for Urban Informatics and Visualization

Start with running the data_aquisition notebook, this will let you get the datasets for LODES8 2023, ACS, and MTC's EPC data for the Bay Area. 

Running the day_night_pop notebook will allow generation of the daytime population map, the nighttime population map, and the net change in population map.

Running the summary_statistics_visualization notebook will allow you to create a chart comparing county daytime vs. nighttime population.
